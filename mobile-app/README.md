# Mobile App

## Installation

```bash
npm install
```

## Développement

```bash
npm start
# ou
expo start
```

## Build

```bash
expo build:android
expo build:ios
```

## Structure

- `src/screens/` - Écrans de l'application (anciennement pages)
- `src/navigation/` - Configuration de la navigation (anciennement routes)
- `src/components/` - Composants réutilisables
- `src/services/` - Services et utilitaires
- `src/contexts/` - Contextes React
- `src/hooks/` - Hooks personnalisés
- `assets/` - Images et ressources statiques
